Import from CSV

Upload a CSV file (in Excel: File → Save As → CSV UTF-8). First row must be headers. Required columns: Property Name, Owner (or Property Owner Name), Phone 01 (or Phone). Optional: Phone 02, IC Number. Up to 50,000 rows per file. For very large datasets (hundreds of thousands or millions of rows), use the bulk import script — see project scripts/bulk-import-properties.js and README.
